## üöÄ Installation

### Prerequisites

- PHP 8.2 (XAMPP)
- Composer
- Node.js and npm

### Steps

1. Clone the repository:
    ```sh
    git clone https://github.com/LeMinhHoang1204/be_bakery_store.git
    ```
2. Install PHP dependencies:
    ```sh
    composer install
    ```
3. Install JavaScript dependencies:
    ```sh
    npm install
    ```
4. Start project:
    ```sh
    php artisan serve
    ```
    ```sh
   npm run dev
    ```
